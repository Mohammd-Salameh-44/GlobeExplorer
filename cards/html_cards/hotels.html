Hotels

The Hotels feature provides an overview of accommodation options available in the selected city. It enables users to explore hotel-related information before making travel arrangements.

Hotel data is collected using the Wikipedia API by searching for pages associated with hotels in the chosen city. The system retrieves summarized information about the most relevant results.

Each hotel entry includes a short description and a link to the original Wikipedia page, allowing users to access more detailed information when needed.

This feature assists travelers in making informed decisions about lodging options and enhances the overall planning experience by centralizing travel-related information in one place.
