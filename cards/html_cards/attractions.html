Attractions

The Attractions feature allows users to discover the most popular landmarks, historical sites, and tourist destinations in the selected city. It helps users gain a better understanding of what the city has to offer.

The system retrieves attraction information using the Wikipedia public API. It searches for relevant pages related to the city's attractions and selects the top results based on relevance.

For each attraction, the system extracts a summarized description and provides a direct link to the full Wikipedia article. This allows users to read additional details if they are interested in learning more.

By integrating external knowledge sources, this feature enriches the user experience and supports travel planning by offering educational and cultural insight into the selected destination.
