GlobeExplorer

Map

The Map feature visually presents the geographic location of the selected city using an embedded OpenStreetMap interface. It allows users to see the exact position of the city on a map.

The system first uses the Nominatim Geocoding service to convert the city name into geographic coordinates, specifically latitude and longitude values. These coordinates are essential for accurately locating the city.

After retrieving the coordinates, the application dynamically generates an embedded map with a marker indicating the city’s exact location. Users can interact with the map and explore surrounding areas.

This feature enhances spatial understanding and provides a visual context for the city, making the application more interactive and informative.
